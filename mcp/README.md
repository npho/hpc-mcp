# MCP Servers

A collection of MCP servers in support of HPC environments.
- slurm: For the Slurm job scheduler in lieu of running `sacct` or similar commands yourself.
- gpfs: For IBM StorageScale commands to understand user or group level storage use (e.g., files, inodes, quota).
- prometheus: If your site uses prometheus to monitor the infrastructure, this can be queried to correlate with Slurm accounting data.
